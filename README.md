# VServ UI — Service Marketplace Frontend

VServ UI is a Next.js/TypeScript application powering a multi-vertical service marketplace platform (ReserveHubs). It enables customers to discover, compare, and book appointments with local businesses across multiple industries — restaurants, health services, automotive, and financial services. The platform supports dual personas: end-users browsing and booking services, and business owners managing their profiles, listings, appointments, and reviews.

The application features a comprehensive component library with search/filter capabilities, business profile management, appointment scheduling, review systems, and an admin panel for platform-wide operations. Built with server-side rendering for SEO and performance, it handles complex multi-step workflows for both B2C and B2B interactions.

## Features & Modules

| Module | Description |
|--------|-------------|
| **Multi-Vertical Search** | Browse restaurants, health, automotive, finance services |
| **Business Profiles** | Full business pages with info, hours, photos, reviews |
| **Appointment Booking** | Schedule and manage service appointments |
| **User Authentication** | Sign-in, registration, profile management |
| **Business Dashboard** | Manage listings, appointments, notifications, ratings |
| **Admin Panel** | Platform admin for business listings and customers |
| **Review System** | Rate and review businesses with overall ratings |
| **Favorites** | Save and manage favorite businesses |
| **Ask Me** | Q&A feature for business inquiries |
| **Landing Pages** | Marketing pages with benefits and CTAs |
| **Responsive Components** | Cards, breadcrumbs, navbars, modals, tabs |

## Tech Stack

- Next.js (SSR/SSG)
- TypeScript
- CSS Modules
- Docker (deployment)
- REST API integration (VServ Core backend)
