## Getting Started

Installation Guide:

1. Install node js(node-v14.15.0-x64) <br />
  URL: https://nodejs.org/ja/blog/release/v14.15.0/  <br />
  npm --version : 6.14.8 <br />
  node --version: v14.15.0 <br />
  
2. Install visual studio code <br />
  URL: https://code.visualstudio.com/download
  
3. React Version: <br />
   @react 18
   
4. UI-Setup  <br />
  a. Git clone 
    HTTPS: <br />
    https://github.com/reserve-hubs/ui.git <br />
    SSH:  <br />
    git@github.com:reserve-hubs/ui.git <br />
  b. Open project in visual studio <br />
  c. Open terminal <br />
  d. Execute command <br />
  e. npm install <br />
  f. npm run build <br />
  g. npm run start (production mode) <br />
  h. npm run dev (development mode) <br />
  i. http://localhost:3000/ <br />
  j. npx kill-port 3000 <br />

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Project starts from `pages/index.tsx`.
